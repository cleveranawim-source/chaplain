# Chaplain

기독교 학교 종교 과목 교과세부특기사항 초안 작성 도우미입니다.

## 기능

- 개별 학생 세특 초안 생성
- 종교 일반 카테고리 + 사회정서교육(정체성, 자기 발견, 회복탄력성, 공감 등) 주제 14종
- 주제별로 종교적 관점과 청소년의 삶·성장을 연결하는 설명 문장 자동 삽입
- 반 전체 엑셀 명단 업로드
- 학생별 주제, 활동, 강점, 문장 구조(도입·유형·강도·마무리) 분산 생성
- 중복 표현 검사(비슷한 문장 쌍, 반복 마무리)
- 500자 초과 경고와 내면 신앙 평가 표현 점검
- 교육청 양식에 맞춘 XLSX 결과 저장

## 사용

`index.html` 파일을 브라우저에서 열어 사용합니다. 입력한 내용은 외부 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.
