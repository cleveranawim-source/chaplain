# Chaplain

기독교 학교 종교 과목 교과세부특기사항 초안 작성 도우미입니다.

## 기능

- 개별 학생 세특 초안 생성
- 종교 일반 카테고리 + 사회정서교육(정체성, 자기 발견, 회복탄력성, 공감 등) 주제 14종
- 주제별로 종교적 관점과 청소년의 삶·성장을 연결하는 설명 문장 자동 삽입
- 학년(1/2/3)별 차별화: 같은 주제라도 1학년은 기초 이해·호기심, 2학년은 연결·적용·심화, 3학년은 종합·주체적 해석 관점으로 서술
- 반 전체 생성 시 학년 지정: 파일 학년 열 자동 인식 또는 반 전체를 특정 학년으로 일괄 지정 선택
- 반 전체 엑셀 명단 업로드(학년 열 자동 인식: 1, 3학년, 고3, 3-5, 2학년 5반 등. 못 읽으면 폼 선택을 쓰고 표에 기본값으로 표시)
- 학생별 주제, 활동, 강점, 문장 구조(도입·유형·강도·마무리) 분산 생성
- 일괄 생성 목표 글자수 지정(250~500자)
- 중복 표현 검사(비슷한 문장 쌍, 반복 마무리)
- 500자 초과 경고와 내면 신앙 평가 표현 점검
- 나이스에서 받은 원본 XLSX의 시트명·서식·구조를 그대로 유지한 채 세특 열만 채워 저장(나이스 재업로드 호환)
- 붙여넣기 작업 시에는 기본 양식의 XLSX로 저장

## 사용

`index.html` 파일을 브라우저에서 열어 사용합니다. 입력한 내용은 외부 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.
